# ReachRight Marketing Website

## Project Info

**Live Site**: https://reachrightmarketing.com

## Development

### Prerequisites

- Node.js & npm - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Getting Started

```sh
# Navigate to the project directory
cd website

# Install dependencies
npm i

# Start the development server
npm run dev
```

### Build for Production

```sh
npm run build
```

## Tech Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Deployment

The site is deployed via GitHub Pages with a custom domain configured via the `CNAME` file.
